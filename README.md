# Progress bar  

> Simple & sexy progress bar for `bash`, give it a duration and it will do the rest.

![progress-bar.sh in action](./preview.gif)

### Usage

1. Copy to your code or `source ./progress-bar.sh`
2. Call

```
progress 5
```
